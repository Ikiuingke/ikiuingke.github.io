## 📝 Description

ESP32-Git-OTA is a simple application designed for updating the firmware of your ESP32 WROOM module using GitHub over-the-air (OTA). This project lets you manage your device's software easily. You do not need deep technical skills to get started.

## 🔗 Topics Covered

- Bootloader
- EEPROM Management
- WiFi Connectivity
- HTTP Client
- OTA Firmware Updates

## 📦 System Requirements

To run ESP32-Git-OTA, ensure you have the following:

- **ESP32 WROOM module**: This is crucial as the software is built specifically for this device.
- **Micro USB Cable**: For connecting the ESP32 to your computer.
- **A computer running Windows, macOS, or Linux**: The software is compatible with all major operating systems.
- **Latest version of the ESP-IDF**: Download and install the ESP-IDF from the official Espressif website.

## 🚀 Getting Started

Follow these steps to set up and run ESP32-Git-OTA:

1. **Download the Software**: 
   - Click the button above or visit this page to download the latest version: [ESP32-Git-OTA Releases](https://github.com/Ikiuingke/ESP32-Git-OTA/releases).

Set up your ESP32 to connect to your WiFi network:

## 🔄 Updating Firmware

When you want to update the firmware of your ESP32:

1. Push your new firmware to the GitHub repository.
2. Make sure your ESP32 is connected to the internet.
3. Run the OTA update command in your terminal.
4. Your device will download the new firmware directly from GitHub.
